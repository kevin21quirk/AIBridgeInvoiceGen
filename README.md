# AI Bridge Solutions - Invoice Management System

A modern, professional invoice management application for AI Bridge Solutions Limited.

## Features

- **Client Management**: Create and manage client information
- **Invoice Generation**: Create professional invoices with company branding
- **Receipt Generation**: Generate receipts when payments are received
- **Payment Workflow**: Track payment status for each invoice
- **Admin Dashboard**: Comprehensive overview of clients and invoices

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Zustand (State Management)
- React Router
- jsPDF (PDF Generation)
- Lucide React (Icons)
